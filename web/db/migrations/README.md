# Forge Database Migrations

These files are generated by Drizzle and applied with:

```bash
cd web
npm run db:migrate
```

Use this workflow for schema changes:

1. Edit `web/db/schema.ts`.
2. Run `npm run db:generate -- --name short_change_name`.
3. Review the generated SQL.
4. Run `npm run db:migrate`.
5. Run the test/build checks before committing.

The `meta` folder is Drizzle bookkeeping. Do not hand-edit it during normal
development.

For a fuller explanation, see
[`docs/database-migrations.md`](../../../docs/database-migrations.md).
